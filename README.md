# 🛒 MERN E-Commerce App

A full-stack E-Commerce Web Application built using the MERN Stack (MongoDB, Express.js, React.js, and Node.js). This project provides a complete online shopping experience with user authentication, product management, shopping cart functionality, and an admin panel.

## 🚀 Live Demo

https://mern-ecommerce-app-gules.vercel.app

## 📂 GitHub Repository

https://github.com/Sanjanachauhan764/mern-ecommerce-app

## ✨ Features

### User Features

* User Registration & Login
* JWT Authentication
* Product Listing
* Product Search
* Add Products to Cart
* Remove Products from Cart
* Responsive User Interface

### Admin Features

* Admin Login
* Add New Products
* Update Existing Products
* Delete Products
* View Total Products

## 🛠️ Tech Stack

### Frontend

* React.js

### Backend

## 📦 Installation

### Clone Repository

git clone https://github.com/Sanjanachauhan764/mern-ecommerce-app.git

### Frontend

cd front-end
npm install
npm run dev

### Backend

cd back-end
npm install
node index.js
